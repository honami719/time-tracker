# ⏱ Time Tracker - WEBマーケター向け工数管理ツール

WEBマーケター向けのシンプルな時間・工数管理ツールです。  
作業ログをクライアント別・タスク別に記録し、ダッシュボードで可視化できます。

## 機能

- 📋 作業ログ記録（日付・クライアント・タスク・開始/終了時間・メモ）
- 👥 クライアント管理
- ✅ タスク管理（カテゴリ別）
- 📊 ダッシュボード（日別・クライアント別・タスク別グラフ）
- ☁️ Google スプレッドシートへの自動保存（GAS連携）

## セットアップ

### 1. Google Apps Script の設定

1. [Google スプレッドシート](https://sheets.google.com) を新規作成
2. `拡張機能` → `Apps Script` を開く
3. `gas_script.js` の内容をすべてペーストして保存
4. `デプロイ` → `新しいデプロイ` → `ウェブアプリ`
5. アクセスできるユーザー: **全員** に設定してデプロイ
6. 表示された **URL** をコピー

### 2. app.js のURL設定

```js
// app.js の先頭付近
const GAS_URL = 'ここにコピーしたURLを貼り付け';
```

### 3. ブラウザで開く

`index.html` をブラウザで開けば使えます。

GitHub Pages で公開している場合は、そのURLにアクセスするだけです。

## 使用技術
